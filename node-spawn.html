Spawn in Node.js

Spawn is a child-process API for running shell commands or executables in a more controlled and stream-friendly way.

1. Why spawn is popular

Spawn gives you a stream-based way to handle real-time output from commands, which is especially useful for long-running jobs.

Streaming

Read stdout and stderr incrementally as the process runs.

Control

Pass arguments and process options explicitly.

Reliability

Good for long-running tasks, build tools, and automation.

2. Example

const { spawn } = require('child_process');

const child = spawn('python3', ['--version']);
child.stdout.on('data', (chunk) => process.stdout.write(chunk));
