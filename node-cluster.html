Cluster in Node.js

The cluster module lets you run multiple Node processes on a machine so your app can use more than one CPU core.

1. Why clustering helps

A single Node process can only use one core for CPU-bound JavaScript work. Clustering creates multiple worker processes so traffic can be shared across them.

Master process

Spawns and manages the worker processes.

Worker process

Handles a portion of incoming traffic.

Shared port

Workers listen on the same port via the OS.

2. Example

const cluster = require('cluster');
const os = require('os');

if (cluster.isPrimary) {
  for (let i = 0; i < os.cpus().length; i += 1) {
    cluster.fork();
  }
} else {
  require('./server');
}
